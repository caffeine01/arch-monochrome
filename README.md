# System Info
* Distro: Arch
* Shell: Fish
* Terminal Emulator: Kitty
* Text Editor: neovim/vim
* File Manager: Thunar
* Menus: Rofi
* Web Browser: Firefox
* WM/DE: BSPWM
* GTK Theme: Custom
